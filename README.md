# Crop Yield Prediction & Recommendation System

## Overview

This project uses Machine Learning to help farmers make informed agricultural decisions.

The system provides:

* Crop Recommendation based on soil and weather conditions
* Crop Yield Prediction
* Agriculture Analytics Dashboard
* Downloadable Prediction Reports

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-Learn
* Plotly

## Machine Learning Models

### Crop Recommendation

* Random Forest Classifier
* Accuracy: ~99%

### Yield Prediction

* Random Forest Regressor
* R² Score: 0.8245

## Datasets

1. Indian Crop Production Dataset
2. Crop Recommendation Dataset

## Features

* Crop Recommendation
* Yield Prediction
* Dashboard Analytics
* Download Reports
* Interactive Visualizations

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app/app.py
```
